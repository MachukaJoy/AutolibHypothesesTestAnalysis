# AutolibHypothesesTestAnalysis
### Introduction
This project seeks to investigate the electric (bluecars) car usage in Paris by performing Hypothesis Test to see if there is a difference in the means of blue cars taken from two different postal codes selected strategically i.e 75015 and 75017 on weekdays. 

### Motivation
This project was done as part of an assessment for a Data Science Course

### Summary of Analysis
1. Defining the Question
2. Reading the Data.
3. Checking the Data.
4. Performing Univariate, Bivariate
5. Creating Visualizations.
6. Hypthesis Testing
7. Summary and Conclusions.
